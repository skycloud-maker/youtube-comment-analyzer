# product-specs/pdf-export.md

## 기능 개요
대시보드의 주요 분석 결과를 임원 보고용 PDF로 출력하는 기능.
사용자가 포함할 섹션을 선택하고, 현재 필터 상태 기준으로
한글 폰트가 정상 적용된 PDF를 다운로드할 수 있다.

## 사용자 & 목적
- **경영진 보고 담당자**: 월간 VoC 보고서를 PDF로 추출
- **제품 기획팀**: 특정 제품·기간 분석 결과를 공유

## UI 명세

### PDF 출력 버튼
- 위치: 대시보드 우측 상단 (엑셀 다운로드 버튼 옆)
- 버튼명: "보고서 PDF 출력"
- 클릭 시: 섹션 선택 모달 팝업

### 섹션 선택 모달
```
[ PDF 출력 설정 ]

포함할 섹션을 선택하세요:
☑ KPI 요약 (숫자 카드)
☑ 주차별 긍정/부정 차트
☑ CEJ 단계별 신뢰도 테이블
☑ 대표 댓글 모음
☑ 그래프/차트 이미지

출력 기준: 현재 필터 상태 적용
           (국가: KR / 브랜드: LG / 기간: 최근 8주)

[ 취소 ]  [ PDF 생성 ]
```

## PDF 구조 (페이지 순서)

```
Page 1 — 표지
  - 리포트 제목: "가전 VoC 분석 보고서"
  - 출력 기준: 필터 상태 (국가/브랜드/제품/기간)
  - 생성 일시

Page 2 — KPI 요약
  - 분석 댓글 수 / 유효 댓글 수 / 제외 댓글 수
  - 긍정 비율 / 부정 비율 / 중립 비율
  - KPI 의견 수 / KPI 긍정 비율 / KPI 부정 비율

Page 3 — 주차별 긍정/부정 차트
  - Altair 차트 → 이미지로 변환 후 삽입
  - 캡션: 기간 범위 명시

Page 4 — CEJ 단계별 신뢰도 테이블
  - 전체 CEJ 테이블 (인지~기타)
  - 신뢰도 지수 공식 캡션 포함

Page 5 — 대표 댓글 모음
  - 긍정 대표 댓글 N개
  - 부정 대표 댓글 N개
  - 원문 + 한국어 번역 병기

(선택 안 한 섹션은 자동 제외)
```

## 기술 명세

### 구현 방식
- **라이브러리**: `weasyprint` (HTML → PDF 변환)
- **대안**: `reportlab` (직접 PDF 생성, 한글 처리 안정적)
- **차트 이미지 변환**: Altair → `altair_saver` 또는
  `selenium` 기반 PNG 캡처 후 삽입
- **한글 폰트**: NanumGothic.ttf 반드시 임베딩
  (미적용 시 한글 전체 깨짐)

### 처리 흐름
```
1. 사용자 섹션 선택
        ↓
2. 현재 필터 상태 스냅샷
        ↓
3. 차트 → PNG 이미지 변환
        ↓
4. HTML 템플릿에 데이터 바인딩
        ↓
5. weasyprint로 PDF 렌더링
        ↓
6. st.download_button으로 전달
```

### 파일명 규칙
```
voc_report_{브랜드}_{YYYYMMDD}.pdf
예) voc_report_LG_20260331.pdf
```

## 폰트 처리 규칙
- PDF 내 모든 한글 텍스트: NanumGothic 적용
- `fonts/NanumGothic.ttf` 경로 하드코딩 금지
  → `_get_font_file()` 유틸 함수 재사용
- 폰트 로드 실패 시: 생성 중단 + 에러 메시지 표시
  ("한글 폰트를 찾을 수 없어 PDF를 생성할 수 없습니다")

## 제약 사항
- Streamlit Cloud에서 `weasyprint` 의존성
  (cairo, pango 등 시스템 라이브러리 필요)
  → `packages.txt`에 시스템 패키지 명시 필요
- 차트 이미지 변환은 서버 환경에 따라 별도 설정 필요
- 대표 댓글 수 너무 많으면 페이지 수 급증
  → 섹션당 최대 10개로 제한
