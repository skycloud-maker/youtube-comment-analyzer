# product-specs/competitor-comparison.md

## 기능 개요
자사 채널과 경쟁사 채널(국내+해외)의 댓글 감성·키워드·
신뢰도를 나란히 비교하여 브랜드 포지셔닝을 파악하는 화면.

## 사용자 & 목적
- **경영진**: "우리가 경쟁사 대비 어디서 앞서고 뒤처지는가"
- **제품 기획팀**: 경쟁사 제품의 불만 키워드 → 자사 스펙 개선 힌트

---

## 비교 대상 채널

| 구분 | 브랜드 | 지역 | 비고 |
|------|--------|------|------|
| 자사 | (자사 브랜드 2~3개) | KR/US | 환경변수로 관리 |
| 국내 경쟁사 | LG | KR | |
| 국내 경쟁사 | Samsung | KR | |
| 해외 경쟁사 | GE | US | 영어 댓글 |
| 해외 경쟁사 | Whirlpool | US | 영어 댓글 |

> 채널 목록은 `BRAND_FILTER_OPTIONS` 상수로 관리
> 추가/제거 시 코드 수정 없이 상수만 변경

## 화면 명세

### 비교 기준 필터
- 제품 카테고리 (냉장고 / 세탁기 / 건조기 등)
- 기간 (주차 단위)
- 비교 브랜드 선택 (자사 고정 + 경쟁사 1~3개 선택)

### 섹션 1 — 브랜드별 감성 비율 비교 (바 차트)
```
브랜드    긍정    부정    중립
자사      72%     18%     10%
LG        65%     25%     10%
Samsung   68%     22%     10%
GE        55%     35%     10%
```
- 브랜드를 X축, 긍정/부정/중립을 누적 바로 표시
- 색상: 긍정 `#4D79F6` / 부정 `#FFAA3B` / 중립 `#A0AEC0`
- 자사 브랜드 바는 테두리 강조 표시

### 섹션 2 — 브랜드별 고객 신뢰도 지수 비교
- 브랜드별 신뢰도 지수 숫자 카드 나란히 표시
- 자사 vs 최고/최저 경쟁사 차이값 강조
  예) "자사 신뢰도 72 — LG 대비 +7p"

### 섹션 3 — 브랜드별 부정 키워드 비교 테이블

| 브랜드 | 1위 키워드 | 2위 | 3위 | 4위 | 5위 |
|--------|-----------|-----|-----|-----|-----|
| 자사 | 소음 | 가격 | AS | 배송 | 설치 |
| LG | 가격 | 소음 | 무게 | AS | 앱 |
| Samsung | AS | 소음 | 가격 | 디자인 | 설치 |

- 자사와 경쟁사가 공통으로 가진 키워드 강조 표시
  (업계 공통 불만 vs 자사 단독 불만 구분)

### 섹션 4 — CEJ 단계별 경쟁사 비교 히트맵
- 행: 브랜드
- 열: CEJ 단계 (인지~교체)
- 셀값: 해당 단계 신뢰도 지수
- 목적: "삼성은 배송 단계가 약하고, 우리는 AS 단계가 약하다"
  같은 구간별 취약점 비교

### 섹션 5 — 시계열 감성 트렌드 비교 (라인 차트)
- X축: 주차
- Y축: 긍정 비율
- 브랜드별 라인 (색상으로 구분)
- 목적: 특정 시점 이후 경쟁사 감성이 급락/급등한 이유 추적

## 다국어 처리 원칙
- 해외 채널(GE, Whirlpool) 댓글은 영어 원문 수집
- 한국어 번역 후 감성 분석 적용
- 키워드 추출은 번역된 한국어 기준
- 원문 댓글 보기 시 영어 원문 + 한국어 번역 병기

## 제약 사항
- 해외 채널 댓글 수가 국내보다 많아
  단순 수치 비교 시 왜곡 가능
  → 비율(%) 기준으로만 비교, 절대값 병기 시 주의 안내
- 브랜드별 영상 업로드 주기가 달라
  동일 기간 내 댓글 수 차이 발생 가능
  → 댓글 수 N 표시로 신뢰도 맥락 제공
