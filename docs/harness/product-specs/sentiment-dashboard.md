# product-specs/sentiment-dashboard.md

## 기능 개요
유튜브 댓글의 감성(긍/부정/중립)을 필터링·시각화하여
제품·브랜드·기간별 고객 반응을 한눈에 파악하는 핵심 화면.

## 사용자 & 목적
- **제품 기획팀**: 제품별 부정 키워드 파악 → 스펙 개선 반영
- **경영진**: 월간 브랜드 감성 트렌드 확인

## KPI 카드 명세

### Row 1 — 댓글 현황 (3개)
| 카드명 | 계산식 | 단위 |
|--------|--------|------|
| 분석 대상 댓글 | 필터 적용 후 전체 댓글 수 | 건 |
| 유효 댓글 수 | 전체 - 제외 댓글 | 건 |
| 제외 댓글 수 | removed_reason 있는 댓글 | 건 |

### Row 2 — 감성 비율 (3개)
| 카드명 | 계산식 |
|--------|--------|
| 긍정 비율 | 긍정 수 ÷ (긍정+부정+중립) × 100 |
| 부정 비율 | 부정 수 ÷ (긍정+부정+중립) × 100 |
| 중립 비율 | 중립 수 ÷ (긍정+부정+중립) × 100 |

### Row 3 — 의견 단위 모니터링 (5개)
| 카드명 | 계산식 |
|--------|--------|
| 원천 콘텐츠 수 | 원본 댓글 수 |
| 의견 단위 수 | 분리된 opinion_unit 수 |
| 분리 비율 | (의견단위 - 원천) ÷ 원천 |
| 중립 비율 | 중립 의견 ÷ 전체 의견 |
| 제외 비율 | 제외 의견 ÷ 전체 의견 |

### Row 4 — KPI 요약 (3개, 임원용)
| 카드명 | 계산식 |
|--------|--------|
| KPI 의견 수 | 긍정 + 부정 수 |
| KPI 긍정 비율 | 긍정 ÷ KPI 의견 수 |
| KPI 부정 비율 | 부정 ÷ KPI 의견 수 |

## 차트 명세

### 1. 주차별 긍정/부정 응답률 (바 차트)
- **라이브러리**: Altair
- **X축**: 주차 시작일 (축약 표기, -35도 회전)
- **Y축**: 응답률 (% 포맷)
- **색상**: 긍정 `#4D79F6` / 부정 `#FFAA3B`
- **바 크기**: 주차 수에 따라 동적 조정 (8~16px)
- **툴팁**: 주차 시작일 / 감성 / 댓글 수 / 응답률
- **페이지네이션**: 8주 단위, 전체 페이지 수 캡션 표시

### 2. 부정 키워드 비중 패널
- 필터 조건 내 부정 댓글에서 상위 키워드 추출
- 비즈니스 무관 키워드 자동 제거 (`filter_business_keywords`)
- 키워드 정규화 적용 (`normalize_keyword`)
- 표시 개수: 상위 6개

### 3. 긍정 키워드 비중 패널
- 부정 키워드 패널과 동일 구조, 긍정 댓글 대상
